About Me


Le programme de la formation developpeur web: Lors de cette formation developpeur web, apprenez les notions fondamentales en JavaScript, maîtrisez les bases algorithmiques de JavaScript, manipulez les événements pour apporter de l'interactivité à l’UI, et comprenez les notions de DOM, de boucles et de conditions pour créer un site web dynamique.

Vous verrez les balises fondamentales en HTML, le CSS et le positionnement flexbox, le responsive web design. Vous maîtriserez le métier développeur web, d'un développeur intégrateur ou d'un développeur Frontend.
Vous apprendrez à coder côté Back-end et maîtriserez l'environnement Node.js, la mécanique du framework Express et la dynamisation de vos pages avec le moteur de template EJS.

Pendant ce Coding Bootcamp, vous verrez la mise en place et le fonctionnement d’une base de données.
Vous allez travailler avec MongoDB, apprendrez à créer une base de données non relationnelle et écrirez vos requêtes NoSQL pour réaliser des opérations CRUD.
